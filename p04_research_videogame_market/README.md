# Исследование рынка видеоигр. Анализ факторов влияющих на успех игры

## Задача

Имеется набор исторических данных с 1980 по 2016 год с информацией о продажах видеоигр, оценками пользователей и критиков, жанром и платформой. **Необходимо выяснить какие факторы влияют на успешность видеоигры.** Заказчик - компания, занимающаяся продажами игр по всему миру. Результаты анализа позволят планировать стратегию реализации новых игр и рекламные бюджеты.

## Библиотеки и методы

`pandas` `seaborn` `scipy.stats`

- Обработка отсутствующих значений и дубликатов;
- Создание сводных таблиц;
- Анализ данных;
- Визуализация данных;
- Создание сводных таблиц;
- Проверка статистических гипотез.

## Описание решения

Представленные данные собраны автоматическим способом из базы данных. Таблица содержит пропуски и дубликаты. 

1. **Предобработка данных**. На первом этапе заполнил пропуски, удалил неинтерпретируемые записи и полные дубликаты. Создал новый признак - общая сумма продаж во всех регионах.

2. **Анализ данных**. В результате анализа зависимостей различных параметров выявил ряд закономерностей:
    - Провел анализ релизов для различных платформ, определил "время жизни" платформы в годах. С помощью этого параметра выделил круг актуальных платформ и предположил характер их развития на будущий год. На графиках сравнил продажи игр для актуальных платформ, выделил из них перспективные платформы, на которые следует обратить внимание.
    - Построил графики распределений и зависимостей ряда параметров игр и продаж. Выделил наиболее популярные жанры, установил влияние оценок экспертов и рейтинга игр на их продажи.
    - По данным продаж создал "портрет" покупателя в разных регионах, определил их предпочтения в жанрах и платформах.

3. **Статистический анализ**. Сформулировал гипотезы на основе вопросов, интересующих заказчика. Провел статистические тесты гипотез и сделал выводы по результатам. Использовал T-тесты для независимых выборок.

4. **Вывод**. Сформулировал вывод по проделанной работе. В выводе перечислил параметры, влияющие на продажи игр: платформы, популярные жанры, оценки критиков, возрастной рейтинг, регион продаж. Полученные результаты можно использовать при планировании деятельности компании.
